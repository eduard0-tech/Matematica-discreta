Conjuntos

O que são Conjuntos?

Um conjunto é uma coleção de objetos distintos chamados elementos. Exemplos:

  • Vogais: A = {a, e, i, o, u}
  • Números pares: B = {2, 4, 6, 8, ...}
Operações Fundamentais:
União (∪): Elementos de A ou B
A ∪ B = {x | x ∈ A ∨ x ∈ B}
Interseção (∩): Elementos comuns
A ∩ B = {x | x ∈ A ∧ x ∈ B}
Complemento (c): Elementos não pertencentes
Ac = {x | x ∉ A}
Subconjunto (⊆): Todos elementos contidos
Se A ⊆ B, então ∀x ∈ A → x ∈ B

Relações

Relações Binárias

Uma relação R em um conjunto A é um subconjunto de A × A. Propriedades essenciais:

  • Reflexiva: ∀x ∈ A, (x,x) ∈ R Ex: ≤ nos números reais
  • Simétrica: Se (x,y) ∈ R → (y,x) ∈ R Ex: Relação de igualdade
  • Transitiva: Se (x,y) ∈ R e (y,z) ∈ R → (x,z) ∈ R Ex: Relação de divisibilidade
  • Equivalência: Quando é reflexiva, simétrica e transitiva Ex: Congruência módulo n

Funções

Teoria das Funções

Entenda os principais conceitos sobre funções matemáticas e suas propriedades fundamentais

Função Injetora

Cada elemento do domínio mapeia para um elemento único no contradomínio

Exemplo: f(x) = 2x
Função Sobrejetora

Todo elemento do contradomínio é imagem de pelo menos um elemento do domínio

Exemplo: f(x) = x³
Função Bijetora

Simultaneamente injetora e sobrejetora, permitindo existência de função inversa

Exemplo: f(x) = x + 3

Função Composta

A combinação de duas funções onde a saída de uma se torna a entrada da outra:

f(g(x)) = (f ∘ g)(x)
  • Domínio deve compatibilizar
  • Não é comutativa

Função Inversa

Função que "desfaz" a operação da função original, existindo apenas para funções bijetoras:

Se f(x) = y, então f⁻¹(y) = x
  • Grágico simétrico
  • Domínio e contradomínio trocados

Tipos de Funções

Uma função f: A → B relaciona cada elemento de A a exatamente um elemento de B.

Injetora

Elementos distintos no domínio têm imagens distintas

Sobrejetora

Todo elemento do contradomínio é imagem de algum elemento

Bijetora

Função simultaneamente injetora e sobrejetora

Função Composta

Dadas f: A → B e g: B → C, a composta g∘f: A → C é definida por:

(g∘f)(x) = g(f(x))

Função Inversa

Se f: A → B é bijetora, existe f⁻¹: B → A tal que:

f⁻¹(f(x)) = x e f(f⁻¹(y)) = y

Sequências Numéricas

Padrões matemáticos fundamentais na organização de números

O que são Sequências?

Sequências numéricas são listas ordenadas de números que seguem um padrão específico. Cada elemento é chamado de termo e sua posição é indicada por um índice.

Termo Geral: Fórmula que calcula qualquer termo
aₙ = f(n)
Classificação: Principais tipos
  • Progressão Aritmética (PA)
  • Progressão Geométrica (PG)
Progressão Aritmética (PA)

Sequência onde cada termo é obtido somando uma razão (r) ao anterior.

Fórmulas:
  • Termo geral: aₙ = a₁ + (n - 1)・r
  • Soma dos termos: Sₙ = (a₁ + aₙ)・n / 2
Exemplo: 2, 5, 8, 11... (r = 3)

Progressão Geométrica (PG)

Sequência onde cada termo é obtido multiplicando o anterior por uma razão (q).

Fórmulas:
  • Termo geral: aₙ = a₁・qn-1
  • Soma dos termos: Sₙ = a₁(qn - 1)/(q - 1)
Exemplo: 3, 6, 12, 24... (q = 2)
